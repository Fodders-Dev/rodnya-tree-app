## Скриншоты

**Это критическая часть и её нужно решить с Артёмом ПЕРВЫМ ДЕЛОМ.** RuStore требует минимум 2 скриншота, рекомендует 5–8, в портретной ориентации не меньше 1080×1920.

В репозитории есть только дев-кадры в `.claude/screenshots/` — они НЕ ПОДОЙДУТ. Это отладочные скрины экранов логина, без полезного контента.

### План А — спросить Артёма прислать новые

Спроси Артёма прислать в чат скриншоты с **его телефона** (там стоит свежий APK) с этих экранов:

1. **Лента / главная** — желательно с парой постов
2. **Дерево семьи** — у Артёма есть «Семья Кузнецовых», она прекрасно подойдёт. Скриншот именно этого экрана с ветками родни.
3. **Список чатов** — главный список разговоров
4. **Чат с человеком** — с историей сообщений, голосовыми, реакциями
5. **Профиль (свой)** — заполненный
6. **Настройки → Внешний вид** — новый theme picker (хвалит свежесть)
7. *(опционально)* Экран звонка во время разговора

Скриншоты должны быть:
- **Portrait, минимум 1080×1920** (на современных телефонах нативный screenshot Volume+Power уже подходит)
- Без debug-overlay (если был включён — выключить в настройках разработчика)
- Без личных данных третьих лиц без их согласия (но «Семья Кузнецовых» — родня Артёма, это нормально)

### План Б — без скриншотов нельзя

RuStore не пропустит листинг без минимум 2 скриншотов. **Если Артём не может прислать прямо сейчас** — останови работу и сообщи: «без скриншотов отправить на модерацию не получится, прислать не позже чем перед нажатием „опубликовать“».

## Если что-то пойдёт не так

- **Залогинило обратно** → скажи Артёму. Не пытайся войти за него.
