# Шпаргалка: две формы РКН (заполнение ~1 час)

> Подготовлено по фактам кодовой базы 2026-06-12. Юр-формулировки сверь с юристом, но поля можно заполнять по этой шпаргалке.

## Форма 1 — Уведомление оператора ПДн
**Где:** pd.rkn.gov.ru → «Подать уведомление» (вход через ЕСИА/Госуслуги, бесплатно).

| Поле формы | Что вписать |
|---|---|
| Оператор | Кузнецов Артём Андреевич, физическое лицо (паспортные данные из ЕСИА подтянутся) |
| Адрес | твой адрес регистрации |
| Цель обработки | Предоставление функций интернет-сервиса «Родня» (rodnya-tree.ru, мобильное приложение): ведение пользователями семейного древа и анкет, обмен сообщениями и звонками, семейная лента и календарь, уведомления |
| Правовое основание | ст. 23, 24 Конституции РФ; ГК РФ; 152-ФЗ; согласие субъектов ПДн; договор (пользовательское соглашение) |
| Категории ПДн | ФИО; дата и место рождения; адрес электронной почты; номер телефона; фотографии; сведения о родственных отношениях; иные данные, предоставленные субъектом (биографические сведения) |
| Специальные категории | не обрабатываются |
| Биометрические | не обрабатываются (фотографии не используются для идентификации) |
| Категории субъектов | пользователи сервиса; родственники пользователей, сведения о которых внесены пользователями |
| Перечень действий | сбор, запись, систематизация, накопление, хранение, уточнение, использование, передача (предоставление обработчикам), блокирование, удаление, уничтожение |
| Способ обработки | автоматизированная, с передачей по сети Интернет |
| Трансграничная передача | **нет** (после переноса серверов в РФ; если подаёшь ДО переноса — юрист: указать Нидерланды временно или ускорить перенос) |
| Сведения о месте нахождения БД | Российская Федерация, [город ЦОД после переноса — впиши из МИГРАЦИЯ_РФ.md] |
| Меры защиты | TLS-шифрование, хэширование паролей, разграничение доступа, журналирование, резервное копирование; назначен ответственный за обработку (сам оператор) |
| Дата начала обработки | дата запуска сервиса [впиши, ориентир — апрель 2026] |

## Форма 2 — Заявление в реестр ОРИ
**Где:** 97-fz.rkn.gov.ru → форма для организатора распространения информации.

| Поле | Что вписать |
|---|---|
| Заявитель | Кузнецов Артём Андреевич, гражданин РФ |
| Наименование ресурса | Сервис «Родня» |
| Доменные имена | rodnya-tree.ru; api.rodnya-tree.ru |
| Описание сервиса | Семейная социальная сеть: обмен личными сообщениями между пользователями, аудио- и видеозвонки, обмен изображениями и записями в семейной ленте |
| Признаки ОРИ | обеспечение приёма, передачи и обработки электронных сообщений пользователей сети Интернет |
| Хостинг | [провайдер РФ после переноса; до — фактический] |
| E-mail для взаимодействия | ahjkuio@gmail.com [рекомендация: завести отдельный ящик на домене, напр. legal@rodnya-tree.ru — позже] |
