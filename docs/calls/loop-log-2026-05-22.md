* New `RodnyaCallForegroundService.kt` (~230 LOC) — Service
  extending android.app.Service. onStartCommand с ACTION_START /

  с extras (singleTask launchMode surfaces existing instance,
  onNewIntent routes через bridge).
* New `RodnyaCallForegroundBridge.kt` (~165 LOC) — MethodChannel
  handler для `rodnya.calls/foreground`. Methods:
  `startCallService(callId, peerName, isVideo, micEnabled)`,
  `updateCallService(...)`, `stopCallService()`,
  `consumePendingNotificationAction()`. Action button taps stored
  в SharedPreferences (mirror pattern `RodnyaTelecomBridge`),
  Dart polls на app resume.

  configureFlutterEngine + onNewIntent (parallel к существующему
  `RodnyaTelecomBridge`).

  "rodnya_active_call"` channel (HIGH importance, no sound,
  no vibration, visibility public, no badge).
* New `lib/services/call_foreground_service.dart` (~120 LOC) —
  Dart client с `start/update/stop/consumePendingNotificationAction`
  методами + `CallForegroundNotificationAction` payload class.
  No-op на non-Android (kIsWeb либо платформа != android).

  `CallForegroundService?` через constructor + field. New helpers
