# RuStore Screenshot Shot List - 1.0.2

Цель: быстро собрать финальный набор Android screenshots без выдумывания контента в последний момент.

## Device Baseline
- Android portrait screenshots
- release build: `rustoreRelease`
- demo dataset: см. secure local file `.tmp/rustore_moderation_credentials_2026-04-12.md`

## Required Set
1. `01-login-or-home.png`
   - Экран: старт после входа или главный экран
   - Что показать: живой продукт, навигацию и бренд `Родня`
   - Подпись: `Соберите дерево семьи и быстро переходите к нужным людям`

2. `02-tree-view.png`
   - Экран: дерево `Семья для модерации RuStore`
   - Что показать: полотно дерева без тяжёлых текстовых блоков, компактные controls
   - Подпись: `Держите семейное дерево в фокусе, а не служебные панели`

3. `03-relative-card.png`
   - Экран: карточка родственника или профиль
   - Что показать: основные данные, быстрые действия и переходы
   - Подпись: `Храните важные данные о родственниках в одном месте`

4. `04-chat-list.png`
   - Экран: список чатов
   - Что показать: поиск, непрочитанное, приватные семейные диалоги
   - Подпись: `Личные и семейные чаты рядом с деревом`

5. `05-chat-media.png`
   - Экран: direct chat demo с медиа-сообщением
   - Что показать: текст, вложение, привычный desktop/mobile chat UX
   - Подпись: `Фото, видео и голосовые сообщения внутри семейного круга`

6. `06-settings-safety.png`
   - Экран: настройки / безопасность / удаление аккаунта
   - Что показать: trust layer, legal/support path
   - Подпись: `Поддержка, жалобы, блокировки и удаление аккаунта без скрытых шагов`

## Capture Notes
- Не использовать debug build, только `rustoreRelease`.
- Перед скриншотами скрыть служебные snackbars и временные error states.
- Для tree screen выбрать масштаб, где видно само дерево, а не пустое полотно.
- Для chat media screen использовать demo chat из moderator dataset.
- Если нужен быстрый экспорт из устройства, сохранять raw PNG в `.tmp/rustore_screenshots_1.0.2/`, а финальный отобранный набор хранить вне git.
